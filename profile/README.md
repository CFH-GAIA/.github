# GAIA Project

## Overview

GAIA is Cathay Financial Holding's strategic initiative to drive digital transformation and innovation across our financial services ecosystem. This GitHub organization hosts the source code, documentation, and resources for the GAIA project.

## About Cathay Financial Holding

Cathay Financial Holding is a leading financial services company in Taiwan, providing comprehensive financial solutions including insurance, securities, banking, and investment services.
